# 哈囉世界！

歡迎來到我的新部落格，它是使用 SvelteKit 和 Markdown 建立的。這是一篇示範部落格如何運作的範例文章。

## 功能特色

這個部落格包含以下功能：

- **Markdown 支援**：使用 Markdown 格式撰寫文章
- **響應式設計**：在所有裝置上都能完美呈現
- **極簡主題**：乾淨簡潔的設計
- **社群分享**：在社群媒體上分享文章
- **分析追蹤**：使用 Umami 追蹤頁面瀏覽量

## 程式碼範例

這是一個簡單的程式碼範例：

```js
function greet(name) {
  return "Hello, " + name + "!";
}

const greetTemplate = (name) => `Hello, ${name}!`;

console.log(greet('World'));
console.log(greetTemplate('World'));
```
